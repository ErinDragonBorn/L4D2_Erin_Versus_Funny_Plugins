# L4D2 药抗趣味插件包 (Add-on for Sirplease)

这是一个基于 [Sirplease 药抗插件包](https://github.com/sirPlease/L4D2-Competitive-Rework) 开发的增强型插件包 。你只需将本项目文件覆盖进原插件包即可使用 。

---

## 🛠️ 使用说明
* **快速安装**：将本项目文件直接覆盖进 Sirplease 的核心文件夹。
* **深度自定义**：本人在配置文件 `cfg/sharedplugins.cfg` 中做了大量注释，方便玩家自行提取插件。

---

## 📑 配置介绍 (点击查看详细参数)

<details>
<summary><b>Zone-WitchParty 女巫派对</b></summary>
<blockquote>

**生还者：** 
* **武器调整**：安全室随机刷新两把近战；移除所有冲锋枪，仅保留单喷（备弹 100）并新增鸟狙（属性同 EQMod）。
* **推限制**：1 推CD。
* **惩罚机制**：当一名生还者倒地时，会立即生成一只 Witch。

**感染者：** 
* **Witch 机制**：解除生成限制，每 10s 生成一只（上限 4 只）；秒杀 Witch 额外得 10 分；距离生还者 1500 码外会被自动处死。
* **特感限制**：仅保留 Hunter 和 Tank，移除其他所有特感。
* **Hunter 强化**：高扑伤害上限解锁至 50。
</blockquote>
</details>

<details>
<summary><b>Zone-Advanced 进阶包抗</b></summary>
<blockquote>

* **物资调整**：将止疼药替换为医疗包。
* **评分机制修改**：采用 **DRDK 评分机制**。
</blockquote>
</details>

<details>
<summary><b>ZoneMod Quad 四控狂欢</b></summary>
<blockquote>

* **特感配置**：移除了 Boomer 和 Spitter，感染者阵营现在最多可同时拥有 2 只 Hunter 。
</blockquote>
</details>

<details>
<summary><b>ZoneMod 1v1 NO Jockey</b></summary>
<blockquote>

* **玩法优化**：移除了 Jockey（猴子） 。
</blockquote>
</details>

<details>
<summary><b>ZoneMod 1v4 练习模式</b></summary>
<blockquote>

* **对抗性质**：类似于药役的 1vN 模式，对手为四名真人玩家。
* **练习功能**：支持无限重开，适合高强度抗压练习。
</blockquote>
</details>
